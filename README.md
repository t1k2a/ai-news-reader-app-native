# AI News Reader

最新のAI関連ニュースを自動収集し、日本語に翻訳して表示するウェブアプリケーションです。

## 機能

- 複数のRSSフィードからAI関連ニュースを自動収集
- 英語記事の日本語への自動翻訳
- カテゴリ別・ソース別フィルタリング
- 無限スクロールによる記事閲覧
- 新着通知機能
- ブックマーク機能
- レスポンシブデザイン

## 技術スタック

- フロントエンド: React, TypeScript, TailwindCSS
- バックエンド: Node.js, Express
- データ処理: RSS Parser, 翻訳API連携

## 最適化機能

- サーバーサイドキャッシュによる高速読み込み
- 翻訳結果のキャッシング
- 効率的なデータ取得と処理
