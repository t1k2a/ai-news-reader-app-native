# 📍 フェーズ1: 投稿の質を改善

## 目的
現在の投稿フォーマットを `social-content` スキルのベストプラクティスに基づいて改善し、エンゲージメントを最大化する。

## 現状の問題点

### 現在の投稿フォーマット
```
{タイトル}

{URL}

#AI #GlotNexus #OpenAI
```

### 致命的な欠陥
1. ❌ **フック（Hook）がない** → スクロールで素通りされる
2. ❌ **価値提案が不明** → なぜクリックすべきか不明
3. ❌ **感情的要素ゼロ** → 共感も好奇心も生まれない
4. ❌ **エンゲージメント要素なし** → いいね・RTする理由がない
5. ❌ **URLがボディに** → アルゴリズムに嫌われる可能性

## 実装タスク

### タスク 1.1: 要約の日本語翻訳機能を追加
**優先度**: 高
**所要時間**: 2-3時間

#### 背景
現在、記事の要約（summary）が英語のままで、日本語ユーザーにとって読みにくい。

#### 実装ステップ
1. `lib/translation-api.ts` を確認し、既存の翻訳機能を把握
2. `lib/rss-feed.ts` で記事取得時に要約を日本語翻訳
3. 翻訳結果を `AINewsItem.summaryJa` などの新フィールドに保存
4. キャッシュ戦略を検討（翻訳APIコスト削減のため）

#### 期待される成果
- 日本語の要約が自動生成される
- 投稿文が日本語ユーザーに理解しやすくなる

#### 関連ファイル
- `lib/translation-api.ts`
- `lib/rss-feed.ts`
- `lib/types.ts` (型定義の拡張)

### タスク 1.2: 強化版投稿フォーマットの適用
**優先度**: 高
**所要時間**: 1-2時間

#### 実装ステップ
1. `lib/auto-post-enhanced.ts` のレビューと改善
2. 要約の日本語版を使用するように修正
3. フックテンプレートを日本語に最適化
4. A/Bテスト用の設定追加（環境変数で切り替え可能に）

#### 改善版フォーマット
```
🚨 {フック}

{日本語要約（80文字程度）}

詳細👇
{URL}

#{ハッシュタグ} #AI #人工知能 #GlotNexus
```

#### フックテンプレート（日本語最適化）
- **好奇心型**: 「🚨 {トピック}が変わった理由」
- **価値提示型**: 「💡 {トピック}で知っておくべき3つのこと」
- **ストーリー型**: 「また{ソース}がやってくれた」
- **速報型**: 「【速報】{ソース}、{トピック}で新展開」

#### ハッシュタグ戦略
```typescript
// ソース別タグ + ジェネリックタグ + ブランドタグ
["#OpenAI", "#AI", "#人工知能", "#機械学習", "#GlotNexus"]
```

最大5個、関連性の高い順に配置。

#### 期待される成果
- スクロール停止率向上（視覚的フック）
- クリック率向上（価値提案の明確化）
- エンゲージメント向上（CTA追加）

#### 関連ファイル
- `lib/auto-post-enhanced.ts`
- `lib/auto-post.ts`

```json
{
  "crons": [

      "path": "/api/cron/auto-post",
      "schedule": "0 7,12,19 * * *"
    }
  ]
}
```

#### 期待される成果
- インプレッション数が20-30%向上
- エンゲージメント率が10-15%向上


   ```json
       "@type": "Organization",
       "name": "GlotNexus"

     "publisher": {
   ```

## 🎯 成果が出ない場合のトラブルシューティング

### ケース1: インプレッション数が増えない
**原因と対策**:
1. **投稿時間が悪い**
   - → 最適時間に変更（7時、12時、19時）
2. **フックが弱い**
   - → フックスタイルを変える（curiosity → value）
3. **ハッシュタグが機能していない**
   - → トレンドタグを追加、数を3-5個に調整
4. **アルゴリズムに嫌われている**
   - → URLを別行に、外部リンクを減らす

### ケース2: エンゲージメント率が低い
**原因と対策**:
1. **内容に共感・価値がない**
   - → 要約を改善、日本語化
2. **CTA（行動喚起）がない**
   - → 質問を追加、意見を求める
3. **フォロワーの質が低い**
   - → ターゲットアカウントへのエンゲージメント強化
4. **投稿頻度が高すぎる**
   - → 1日3回に減らす

### ケース3: クリック数が少ない
**原因と対策**:
1. **価値提案が不明確**
   - → 「詳細👇」以外のCTAも試す
2. **要約で満足されている**
   - → 要約を短くし、続きを気にさせる
3. **URLが長すぎる**
   - → 短縮URLの検討（bit.ly等）

### ケース4: フォロワーが増えない
**原因と対策**:
1. **エンゲージメント活動不足**
   - → 毎日30分のエンゲージメント徹底
2. **プロフィールが魅力的でない**
   - → プロフィール文・アイコンを改善
3. **投稿の一貫性がない**
   - → コンテンツの柱を守る
4. **価値が伝わっていない**
   - → 固定ツイートで価値提案を明確に
