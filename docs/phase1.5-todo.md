## 毎日やること（30分）

### 検索クエリ（曜日ローテーション）

| 曜日 | クエリ |
|------|--------|
| 月・木 | `#生成AI lang:ja` |
| 火・金 | `ChatGPT 使ってみた lang:ja` |
| 水・土 | `Claude 試した lang:ja` |
| 日 | `LLM 日本語 lang:ja` |

手順：
1. X の検索バーに上記クエリを入力
2. 「最新」タブに切り替える
3. 直近1時間以内の投稿を探す
4. リプライを3〜5件投稿する

### リプライの型

❌ NG（効果なし）
```
「参考になります！」
「勉強になりました！」
```

✅ 型A：追加情報を足す
```
GlotNexusで今日この話題の英語原文も読みました。
海外では〇〇という観点での議論が多いですね。
```

✅ 型B：自分の実体験を足す
```
自分も試してみたら△△でした。
特に〇〇の部分が想定外でした。
```

✅ 型C：GlotNexusへ誘導
```
これに関連して、今日OpenAIが〇〇を発表してます。
glotnexus.jp で日本語まとめ読めます
```
